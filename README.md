# ------------------Python-Webhook-transmitter------------------

Hello, i made a Python webhook transmitter set!

The file "custom.py" is for sending messages, but you will always have to paste your webhook url.

The file "main.py" is for sending messages, but always the same. (You have to paste your msg and url in the file)

The file "main_custom.py" is for sending messages, (you have to paste your url in the file, but you can send always diffent mesagges
